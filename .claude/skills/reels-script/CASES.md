**數據**:觀看 未知 / 讚 26,400 / 留言 3,000 / 轉發 未知 / 收藏 未知(取得時間:發布後約 2 週)

**節奏**:未知(僅能就標題與數據分析)

3. **遊戲化字眼「段位」「開外掛」**:把工具熟練度包成遊戲排位,比「6 個技巧」更有身分感與勝負感。帳號名本身(開外掛)也是同一語言體系。

## Case #2 — 2026-07-09

**網址**:https://www.instagram.com/reel/DaS3xUYvkUt/
**影片摘要**:@l_khaisin(中文/馬新圈內容行銷教學)「流量密碼第二名:Share Rate」—— 系列型內容,用 IG 後台「What affects your views」的各項 rate 當系列骨架,一集講一列。**開頭 3 秒**:直接螢幕錄 IG 後台數據排行截圖(Skip/Share/Like/Save/Repost/Comment rate)+ 小圓框自拍當 hook;字卡層即數據本身。主體教「怎麼做出讓人想 share 給某個人的內容」(舉例:結尾字卡點名『男朋友』引發轉傳),片尾預告第三名 Like Rate + 「follow me for more」。
**長度**:42.7 秒|**賭的機制**:收藏 + 追蹤(系列追看);caption 另下留言 CTA 但無效
**數據**(來自影片開頭的 IG 後台截圖,即本片自身成效):Skip rate 36.0% / **Share rate 8.5%** / Like rate 6.2% / Save rate 1.2% / Repost rate 0.2% / **Comment rate 0.0%**;外顯:讚 1.2K、留言 7(發布約 6 天)
**使用的鉤子**:新型態 —— 「平台原生數據截圖」(登錄為 R10)+ 系列排名框架(延續 R9 家族)
**節奏**:快;數據截圖 → 講話特寫 → 中段插別支 reel 當範例(PiP,上方紅黑字卡)→ 講話。約每 2–3 秒一變化,符合節奏規範。字幕:白字 + 關鍵字黃色 highlight、中英夾雜、置中偏下描邊無底框。

**流量歸因分析**:
1. **Hook 用平台數據截圖 = 借權威 + 秀成果**:第一幀不是自我介紹也不是手做字卡,是 IG 官方後台的 rate 排行,天然具體可信,直接點題「這些 rate 就是流量密碼」。驗證 MODEL.md「先秀成果、第一幀強畫面」,並超出既有公式庫 → 新增 R10。
2. **系列化結構鎖追看**:把 IG metrics 表的每一列拆成一集(Skip→Share→Like→Save…),觀眾想看完整套必須追蹤 → 直接餵「follow me for more」。是 Case #1 排名/清單模式的進化(資訊源系列化)。
3. **⚠️ 留言 CTA 對資訊型內容失效**:Comment rate 0.0%、僅 7 則留言,即使 caption 明確下了留言 CTA(「留言你想 share 的影片類型」)。本片實際賭的是收藏/追蹤(教學型),硬套留言 CTA 無效 → 佐證「內容型態決定機制,別錯配 CTA」。

**客群洞察**:內容創作者 / 想長流量的個人品牌經營者,對「拆解平台後台數據」「可複製的流量公式」高度有感;會追系列但不一定留言(資訊型內容留言意願低)。

**模型更新**:
- 新增鉤子公式 R10「平台原生數據截圖」(信心度:中高,外部案例 + 自身數據截圖佐證)
- 「八、已驗證模式」新增:資訊源系列化結構、留言 CTA 錯配警示
- 「七、客群輪廓」新增:內容行銷/流量學習客群
- 版本 v1.1 → v1.2

## Case #3 — 2026-07-09(自有帳號,2026-07-12 數據補完)

**網址**:https://www.instagram.com/reel/DaieO7vSV5I/(@ailifeu,使用者本人帳號)
**影片摘要**:純口播單鏡教學。內容:「學 AI 英文不好是硬傷 → Claude 官網有系列課程還發證照,但全英文很痛苦 → 我把基礎課中文化免費放留言區,看完可拿第一階段證照」。開頭 3 秒:靜止講話 + 白色跑動字幕「英文不好真的是硬傷」,無字卡、無強畫面。
**長度**:43.2 秒|**賭的機制**:(意圖)收藏+追蹤(lead magnet 型),但結尾下了按讚+留言+訂閱三連 CTA
**數據**(2026-07-12 後台截圖):觀看 526 / 觸及 372 帳號 / 平均觀看時間 **6 秒**(片長 43.2 秒,僅 14%)/ **略過率 68.0%** / 分享率 0.3% / 新增粉絲 1 / 讚 8 / 留言 1(自留)。曲線第 1 天衝到 ~470 後即轉平 = 只吃到帳號基礎池(平時 baseline ~400–500),未獲外推。
**使用的鉤子**:弱痛點陳述(近 R1 但無承諾);真正的鉤子(免費中文課+證照)埋在 23 秒
**節奏**:❌ 全程 43 秒單一固定鏡頭、零視覺變化(違反「靜止超過 4 秒=滑走高危」)

**流量歸因分析**(craft 診斷,非數據):
1. **料很強但被埋**:「Claude 官方課程中文化+免費+證照」是硬 lead magnet(收藏+追蹤雙殺),但埋到中段才出現,且被結尾三連 CTA 稀釋。犯了模型「前 3–5 秒先兌現承諾、別鋪陳」。
2. **Hook 三層皆弱**:口白只有痛點無 payoff、畫面靜止、無字卡承諾——完播率預測偏低。
3. **CTA 錯配 + 過多**:資訊型內容硬下留言 CTA(v1.2 已知無效),又按讚/訂閱/留言三連(違反「一支一個 CTA」)。

**客群洞察**:目標客群為中文圈想學 AI 但卡英文門檻者;「免費」「中文化」「證照」是有感字眼(痛點+身分獎勵)。

**數據歸因(2026-07-12 補)**:
1. **craft 診斷被數據證實**:靜止單鏡 43 秒、hook 三層皆弱、真鉤子埋 23 秒 → 4 天僅 8 讚、唯一留言是自己的 = 演算法測試期就判定不外推。與模型「第 3 秒分水嶺」「靜止開頭是死刑」一致。
2. **猛料救不了弱包裝(連兩案驗證)**:免費中文課+證照這種 lead magnet 級素材,包在無承諾的痛點開頭裡,連自家粉絲都沒被留住。與 Case #4 構成同帳號連續模式:「料好、包裝弱、CTA 錯配」。
3. **自己留言補 CTA 無效**:關鍵資訊(看完拿證照)放在留言區而不是 hook,等於藏起來。

**模型更新**:R1 加註「痛點必須配承諾」;已驗證模式新增第 7 條「猛料前置」;版本 v1.3 → v1.4。

**與 Case #4 對照歸因(2026-07-12,兩支後台數據齊)**:
| 指標 | Case #3(常青教學) | Case #4(時效梗) | 結論 |
|------|------|------|------|
| 觀看 | 526(= baseline) | ~1,750(4x) | 題材決定演算法給不給測試流量 |
| 略過率 | **68.0%** | 42.5% | hook 決定留下多少:無承諾痛點 vs 時效話題,差 25 個百分點 |
| 平均觀看 | 6 秒 | 7 秒 | **兩支同天花板 —— 單鏡口播無視覺變化,誰進來都在 6–7 秒離開** |
| 分享率 | 0.3% | 0.3% | 內容都沒有「傳給某人」的設計 |
| 轉粉 | +1 | 0 | offer 埋深/缺 CTA,流量不轉資產 |

→ 帳號瓶頸排序:①主體節奏(6–7 秒魔咒,影響 100% 的片)②hook 承諾(略過率)③互動缺口設計。寫入模型第 9 條,版本 v1.5 → v1.6。
