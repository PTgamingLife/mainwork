# 家庭財務模型

> 這是一個「活的模型」。每次上傳理財工具/策略做學習,就更新此檔。
> 產生月度建議時必須完整讀取本檔,「六、已驗證策略」權重最高,其次通用規則。

**模型版本:v1.0**(2026-07-09 初版:通用穩健家庭財務框架,尚未經實際資料校正)

---

## 一、核心原則(所有建議的底層邏輯)

1. **先守後攻**:順序永遠是 現金流為正 → 緊急預備金 → 還高息債 → 保險保障 → 投資。跳步驟等於在流沙上蓋樓。
2. **現金流 > 帳面資產**:每月「收入 − 支出 = 結餘」必須為正。結餘是所有理財動作的燃料;結餘為負,先修這個,別談投資。
3. **儲蓄率是命脈**:儲蓄率 =(收入 − 支出)/ 收入。目標分級:及格 10%、穩健 20%、快速累積 30%+。提升儲蓄率比提高投資報酬率更可控、更有效。
4. **負債要分好壞**:低利房貸(自住)是好債,不急著還;信用卡/信貸/車貸等高息債是壞債,優先清掉。
5. **保守勝於樂觀**:規劃用「最壞情況」抓(收入中斷、利率上升、大額醫療)。買得起最壞情況,才睡得著。
6. **兩人一致**:夫妻對錢的目標與底線要對齊。共同帳戶管家庭固定開銷,各留一部分自主金,減少摩擦。

## 二、預算框架(分配每月收入)

- **50 / 30 / 20 法則(預設骨架)**:需要 50%(房貸、水電、food、保險、交通)/ 想要 30%(娛樂、旅遊、外食升級)/ 儲蓄投資 20%(含還壞債)。
- 若房貸+固定開銷已吃掉 >55% 收入 → 屬「固定成本偏高」,先從「想要」砍,或設法開源。
- **六個罐子(進階,現金流細分)**:生活必需 55% / 長期儲蓄 10% / 玩樂 10% / 教育成長 10% / 財務自由(投資)10% / 分享捐贈 5%。比例可依家庭調整,重點是「每筆錢都有去處」。
- 記帳分類要少而穩(建議 8–10 類),能持續比精細重要。

## 三、緊急預備金(守的第一道牆)

- 目標:**3–6 個月家庭總支出**存成隨時可動用的現金(活存/數位帳戶高利活存)。
- 雙薪、工作穩定 → 3–4 個月即可;單薪或收入波動大(自營、業務、接案)→ 6–12 個月。
- 這筆錢**不拿去投資**,它的任務是「不用在最糟時機賣資產或借高息債」。
- 動用後,回補優先序高於新增投資。

## 四、房貸與負債策略

- **自住房貸(低利)**:利率若低於長期投資期望報酬,不必急著提前還;但「提前還款 = 無風險地賺到貸款利率」,對保守型或高利率環境是好選擇。取捨看:利率高低、心理負擔、有無更好的錢的去處。
- 決策參考線:房貸利率 **< 2.5%** 傾向「維持月供、閒錢投資」;**> 3.5%** 傾向「加速還款」;中間看個人風險偏好。(利率環境會變,學習模式可更新此線。)
- **房貸月供** 建議 < 月收入 33%;**總負債月付** < 月收入 40%。超過就是負債比紅燈。
- **高息債(信貸/卡循環/車貸)**:最優先清償,用「債務雪崩法」(先還利率最高的)最省利息;要動力則用「雪球法」(先還餘額最小的)。
- 重貸/轉貸:利率能降 > 0.5% 且剩餘年限長,值得評估轉貸(算清楚手續費、綁約、地政規費的回本期)。

## 五、保障與資產配置(守穩了才攻)

- **保險先於投資**:優先補「發生機率低但會擊垮家庭」的缺口——壽險(有房貸/有小孩尤其重要,額度可抓「房貸餘額 + 家庭數年開銷」)、實支實付醫療、重大傷病、意外險。避免用高保費儲蓄險擠壓現金流。
- **投資只用「閒錢」**:投入的是緊急預備金與短期(3–5 年內要用)開銷之外的錢。
- **長期資產配置 > 選股擇時**:以低成本、分散(全球指數型 ETF 為核心)、定期定額、長期持有為骨架。股債比可用「年齡」或風險承受度粗估,再依目標調整。
- **先用制度優惠**:勞退自提(節稅)、公司相對提撥、政府補助、節稅工具,通常 CP 值高於自己選標的。
- 不追明牌、不借錢投資、不 all in 單一標的;報酬愈「保證」愈要懷疑。
