## Case #1 — 2026-07-10 【風格參考】

**來源**:@growithfyn(藍勾認證,類似作者,非自己)|使用者提供 2 張封面截圖(貼文① 1/10「The Content System」、貼文② 1/6「丟給 Claude 幫我剪」)
**為什麼選它參考**:同 niche —— 中文 × AI 工具 × 家長內容創作(貼文②直接示範把影片丟給 Claude 剪),等於是本帳號「工程師奶爸 + AI」人設的時尚精緻版對照組。

**視覺萃取(兩則高度一致,視為同一套模板)**:
- 封面版型:**滿版情境照(真人出鏡)+ 由下往上暗色漸層壓底 + 大字左對齊壓在照片上**。非色塊 icon 圖卡。
- 色系:近黑底 + 情境色(貼文①酒紅劇院椅 / 貼文②暮藍山景與郵輪夜景)+ 白字;低飽和、電影感、premium。約 2–3 色。
- 字體:大標=**白色粗明體(雜誌感)**;英文 eyebrow=**斜體 serif**(「The Content System」「Claude」);內文=白色細黑體。字級階層分明。
- icon/配圖:幾乎不用 icon,全靠真人生活照;留白多。
- 品牌固定元素:左上/左下 `@GROWITHFYN` letter-spaced 浮水印;右上圓角膠囊頁碼 `1/10`;封面底印 `SHARE`/`SAVE`;`滑動看完整方法 →` 箭頭。
- 封面文案公式:貼文①痛點共鳴(C3/C8「還在糾結今天發什麼」)+ 留言換資源(留言「流量」,近 threads H8);貼文②未完成故事反轉(C12「結果它回傳給我這個…→」)。
- 成效訊號:貼文①讚346/留言161/轉發7/**存154**;貼文②讚301/**留言421**/轉發6/存160。→ 明顯是**留言 + 收藏機器**,非衝讚型(存≈讚、留言極高),印證 MODEL 第一章「收藏/留言 > 讚」。

**可借鑑到本帳號的點**:暗色電影感 + 真人出鏡 + 粗明體大標 + 膠囊頁碼 + 封面印 SHARE/SAVE + 未完成故事鉤子。這些直接寫入 MODEL 第五章。
**不採用的點**:她的時尚精緻棚拍與 glam 造型不符本人設 → 轉譯為「工程師奶爸樸實生活感」(帶娃/電腦桌/居家/廁所自拍梗);「留言換資源」近互動誘餌,沿用 threads 判斷慎用。

**模型更新**:
- 校正 MODEL.md 第五章全部四個待校正欄位(①色系→暗色電影感 ②字體→粗明體+斜體serif ③版型→滿版真人情境照 ④icon→少用以照片為主),新增膠囊頁碼 / SHARE·SAVE 品牌元素與人設轉譯註記。版本 v1.0 → v1.1。
- 封面公式信心度**未調整**(風格校正模式只動第五章;公式升降級需本帳號自身流量數據)。
