## Case #1 — 2026-07-12

**評估標的**:個人品牌 / 個人 IP(涵蓋 personal branding、自媒體經營、知識變現;評的是「教人/幫人打造個人品牌」市場)
**五維評分**:P5 / R5 / T4 / C4 / K5
**MVS**:(5×5×4)/(4×5) = 100/20 = **5.0** → 🟡 有機會

**評分依據**:
- P(5):LinkedIn 報告 67% 專業型創作者滿一年零變現;痛點明確「有人看有人讚沒人買」「不知道靠什麼賺錢」;且已有大量人付費買課程/顧問求解(付費意願強)。
- R(5):跨滿所有平台(部落格/Threads/FB/PTT/Dcard/104/YT/Podcast),「人人都能做個人 IP」大眾級論述。
- T(4):2026 仍大量趨勢文、工商時報稱「個人 IP 數位轉型師正夯」;動能穩健但已熱多年,非爆發初期。
- C(4):線上課 NT$38k–68k、單次顧問 3k–15k、包月 30k–100k/月 + 3–12 個月時間成本,解方貴又慢。
- K(5):極度紅海,Zoey(3000+ 學員)、Henry Wen、Daily Ding、Hahow/YOTTA 等巨頭林立,通用大詞難出頭。

**結論與建議**:①需求真實不用懷疑(有人痛+有人付錢);②別做通用大詞會撞巨頭,K 把 MVS 壓在 5.0;③往「特定族群×特定痛點」細分切角走(如工程師/醫療專業/素人接案者第一批客戶),K 可降到 2–3,MVS 有機會進 🟢。
**信心度與盲點**:公開搜尋快篩。最不確定 R 的絕對規模與 K 的真實飽和度(需 Ahrefs/Similarweb 定量);P 訊號最紮實(付費證據足)。

## Case #6 — 2026-07-12(觸發校準 C4;模型升 v1.2 → v1.3)

**評估標的**:個人IP商業變現 + AI(讀法:你自己經營會賺錢的個人IP,在你的專業領域,AI 當放大器——非賣課教變現)
**評分**:P4 / R4 / T5 / C3 / K3 / M4
**MVS**:(4×4×5)/(3×3) × (4/3) = 8.89 × 1.33 = **11.85** → 🟢(四變體最高分,但**有條件**)

**評分依據**:
- P(4):強烈渴望(擺脫以工換酬、專業變收入),真實普遍但屬渴望非急性痛。
- R(4):廣,一人公司+AI 是 2026 顯學;但執行者少於想做者。
- T(5):峰值,OPC 商業模式、AI 賦能一人創業=2026 副業主流敘事。
- C(3):AI 壓縮製作(6 月→3–4 週),但需真本事能賣+持續產出。
- K(3):關鍵差異——在自己垂直領域競爭(各做各的),非全擠「教變現」一坑;但 AI 內卷/商品化(HKU)故不給 2。
- M(4):三層收入(高價服務/數位產品/會員);高價服務粉絲少也能賺;品牌者報價 3.5 倍;新手務實月入幾千→1.5–3 萬(非月入百萬幻覺)。

**與 Case #5 對照(同詞不同序)**:AI×個人IP×變現(教變現)K5/信任崩/🟠陷阱;個人IP商業變現+AI(自己做)K3/信任完整/🟢乾淨。詞序(誰是核心)決定進血海或活路。

**盲點與模型升級**:此標的是「方法/操作模式」不是「市場」——11.85 高分**綁定前提資產**(你底下要有真專業/受眾),缺前提則崩塌;且 AI 降門檻→內卷→護城河(信任/專業深度/獨特角度)才是差異化。→ 新增校準 **C4**(市場 vs 方法),模型升 v1.3。

**結論與建議**:①所有變體裡最該走的路,乾淨真實可長久;②先選真正垂直領域建IP,AI 只當放大器;③變現順序:高價服務→數位產品→會員社群;④建護城河擋 AI 內卷。
**信心度與盲點**:最不確定=「你底下的專業/垂直是什麼」(分數成立前提,非市場問題);T/K/M 訊號紮實。

**實際結果 / 校準**:(待回填)

## Case #5 — 2026-07-12(觸發校準 C3;模型升 v1.1 → v1.2)

**評估標的**:AI × 個人IP × 變現(實務上=「教/幫人用 AI 把個人IP變現」的知識付費/教變現賽道)
**評分**:P5 / R5 / T5 / C3 / K5 / M4
**MVS(原始)**:(5×5×5)/(3×5) × (4/3) = 8.33 × 1.33 = **11.1** → 🟢(但為「陷阱分數」)
**新進者實質**:🟠(無信任資產時)

**評分依據**:
- P(5):「變現」是最尖銳的痛,直打最願付錢處(67% 一年零變現)。
- R(5):巨大,「AI 賺錢/副業/變現」2026 最大主題之一,跨平台+出海。
- T(5):爆發峰值,AI 市場 2941→3759 億美元、知識付費 AI 垂直最快變現。
- C(3):課程 99–499 人民幣到十萬元不等;AI 工具便宜。
- K(5):地表最擠知識付費賽道,「AI 補習班滿天飛」。
- M(4):「教人賺錢」本身超好賺(高毛利課程/社群/教練),但信任崩壞拉低轉換故不給 5。

**盲點與模型升級**:原始高分 11.1 反映的是「既有玩家(早進場+有信任者)」價值,非新進者。市場有大量割韭菜/詐騙/智商稅反噬(PTT「都騙錢的」、Dcard 十萬元一頁式課、資安打詐週報)。→ 新增校準 **C3**(高分紅海+信任崩壞=新進者陷阱;信任=K 裝不下的隱藏門檻;教人賺錢類市場對無名新進者向下折),模型升 v1.2。

**結論與建議**:①別做通用「AI 教你變現」(最深血海+信任糞坑,新人必敗);②要做靠「信任資產」差異化(公開真實數據、可查戰績、專攻小利基、反割韭菜定位);③更乾淨的替代:回 Case #2 AIP——自己用 AI 把垂直個人IP做起來順便變現,勝過賣課教變現;④一句話:🟢 的皮、🔴 的裡,除非你有真本事+信任差異化。
**信心度與盲點**:最不確定「你個人是否已有信任資產/戰績」——決定落在 11.1🟢 或實質 🟠;市場面 P/R/T/K 訊號紮實。

**實際結果 / 校準**:(待回填)

## Case #4 — 2026-07-12(Case #3 的細分切角驗證)

**評估標的**:台語兒歌動畫(AI 製作,公版台語童謠 or 新創,鎖定台灣母語教育場景)
**評分**:P4 / R3 / T4 / C3 / K2 / M=雙叉(2 或 4)
**基底(未計 M)**:(4×3×4)/(3×2) = 48/6 = 8.0
**MVS(依商業模式分叉)**:
- A 純 YouTube 廣告:M2 → 8.0×2/3 = **5.3** 🟡(小受眾×COPPA 低 RPM,死路)
- B 政府補助+學校 B2B+教材授權:M4 → 8.0×4/3 = **10.7** 🟢(金脈)

**評分依據**:
- P(4):文化+政策雙驅動(國家語言發展法、母語流失焦慮、家長想讓孩子學台語);消費端急迫度中等。
- R(3):僅台灣、家庭子集,但加上全台國小本土語部定課+幼兒園機構受眾。
- T(4):強順風。國家語言整體發展方案 2022–2026、111 學年本土語入課綱、政府大力推。
- C(3):比英文難,AI 台語配音/唱歌品質弱(發音/白話字/變調),做「對」有門檻。
- K(2):最大翻盤點,無 Cocomelon 級巨頭;momo 親子台/公視台語台/零散歌單,遠未飽和。
- M(2 或 4):純廣告=小盤×COPPA 幾乎不賺(2);但政府正花錢買——文化部「本土語言創作及應用補助」、教育部補助幼兒園/學校、國家語言方案補助民間影音數位內容製作,走 B2G+B2B 可達 4。

**與 Case #3 對照**:英文兒歌 K5/M2 → 5.0 🟡(陷阱);切成台語後 K5→2、多開政府/學校變現路,B 路達 10.7 🟢。驗證「切巨頭懶得做的縫隙」+ M 為決勝維度。

**結論與建議**:①切角正確,把價值陷阱轉成利基;②別靠 YouTube 廣告(5.3 死路),YouTube 當作品集/流量入口,主收入放政府補助標案/學校通路授權/母語教材/周邊(10.7);③先解決「台語做得對」(找顧問或用教育部台語辭典校音,是最大執行風險也是護城河);④搶 YouTube 原生台語兒歌品牌的先行者紅利。
**信心度與盲點**:最不確定 M 的 B 路可行性(補助/標案申請門檻、撥款規模、B2B 銷售週期需個案驗證);K=2、T=4 訊號紮實。模型 v1.1 無需修改。

**實際結果 / 校準**:(待回填)

## Case #3 — 2026-07-12(觸發模型升級 v1.0 → v1.1)

**評估標的**:YouTube 兒歌動畫頻道(用 AI 製作、採無版權/公版兒歌;faceless 頻道形式)
**評分**:P5 / R5 / T3 / C2 / K5 / **M2(新維度)**
**MVS(v1.0,無 M)**:(5×5×3)/(2×5) = 75/10 = **7.5** → 🟡(假高分)
**MVS(v1.1,含 M/3)**:7.5 × (2/3) = **5.0** → 🟡(實則偏 🟠,對新進單人創作者)

**評分依據**:
- P(5):幼兒內容需求地表最強之一,重複觀看、家長剛需。
- R(5):全球跨語言。Cocomelon 2200 億觀看、被以 30 億美元收購。
- T(3):需求常青但非新紅利;AI 製作切角已成「$1000/day」淘金潮(晚了、擠了)+ COPPA 2.0/GDPR-K 監管收緊逆風。
- C(2):AI 讓製作成本暴跌(Magiclight/AutoDraft,無需錄音室/動畫師)。
- K(5):地表最血海之一,Cocomelon/Pinkfong/Super Simple Songs 數十億級巨頭 + 大量 faceless AI 頻道湧入。
- **M(2)**:COPPA 使 Made for Kids 只能投非個人化廣告,RPM 僅 $0.30–0.50(一般 $5–15)、收入掉 95%、失去會員/Super Chat/超級感謝。業配是主要出路但需要對抗巨頭的規模。

**模型缺陷與升級**:此案暴露 v1.0 只獎勵 R(觀眾多)卻不罰「有需求卻收不到錢」。→ 新增 **M 變現效率乘數(×M/3,中性=3,向下相容)**,模型升 v1.1;並寫入校準 C1(流量≠收入)、C2(淘金潮=T/K 警訊)。

**結論與建議**:①需求是真的,但這是「海量觀看、微薄收入、巨頭壟斷」的價值陷阱,對新進單人創作者不建議正面硬做;②若真要做,別碰廣告分潤當主收入——走**業配 / 自有 IP 周邊 / 導流到 App 或課程 / 多語版鋪量 / Shorts 分潤**;③或改切**大平台懶得做的縫隙**(特定語言市場如台語/客語兒歌、特殊教育、特定文化節慶兒歌),K 與競爭下降。
**信心度與盲點**:M 的變現數據紮實(多來源一致);最不確定的是「自有 IP 周邊/導流」這條替代變現路徑對單人創作者的真實可行性(需個案驗證)。

**實際結果 / 校準**:(待回填)

## Case #2 — 2026-07-12

**評估標的**:AIP = AI × 個人 IP(自創縮寫,零搜尋量;評的是「用 AI 工具打造/經營個人 IP」概念,含 AI 生成內容、AI 分身/數位人、AI 自動化自媒體、一人公司)
**五維評分**:P5 / R4 / T5 / C3 / K4
**MVS**:(5×4×5)/(3×4) = 100/12 = **8.33** → 🟢 高潛力(剛跨過 8)

**評分依據**:
- P(5):同個人品牌變現痛 +「AI 讓產製門檻降、信任門檻升」新痛 +「怕跟不上 AI」焦慮;有品牌自由工作者報價 3.5 倍。
- R(4):廣但為個人品牌的 AI 子集;工具/課程/教學量大,略窄於純大詞。
- T(5):最大亮點。AI 虛擬網紅 60 億→458 億美元(2024→2030);「2026 創業新賽道 AI+個人IP」;效率 5–10 倍。
- C(3):AI 本身降成本——每天 30 分鐘 + 數千元課程/工具訂閱即可 DIY,比純個人品牌數萬課程便宜。
- K(4):仍高但比純大詞鬆,AI 切角新、未被壟斷死;但新進者瘋狂湧入,飽和中。

**與 Case #1 對照**:純個人品牌 5.0 🟡 → 加「AI ×」切角後 8.33 🟢。改善來自 T(4→5)、C(4→3)、K(5→4);代價 R(5→4)。驗證了「細分切角能提分」的模型規則 #5。

**結論與建議**:①方向對,AI×個人IP 明顯優於裸做個人品牌;②但是「剛踩線的 🟢」且 K 快速惡化,紅利有時效;③「AIP」自創縮寫雙面刃——零搜尋量=先行者可佔詞但短期無搜尋紅利,得靠內容教育市場;④想推到穩固 🟢 再切一刀 `AI×個人IP×垂直族群`(中小企業主/保險/醫美),MVS 上看 12–15。
**信心度與盲點**:公開快篩。最不確定 K 真實飽和速度(玩家爆增,恐比 4 更擠)與 R 變現轉換率(討論量大≠買單);T 訊號最硬(多份市場規模數據)。

**實際結果 / 校準**:(待回填)
