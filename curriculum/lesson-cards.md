## 每堂課快速速查

### 第 1 堂 | AI 思維破冰
- **核心概念**：AI = 廣度速度，人類 = 深度方向
- **三個層次**：提效 → 轉型 → 創新
- **立即行動**：列出 3 個 AI 可輔助的日常工作

### 第 2 堂 | Prompt 工程
- **RACE 框架**：Role / Action / Context / Example
- **黃金法則**：越具體，輸出越精準
- **迭代技巧**：「請調整 X 讓它更 Y」

### 第 3 堂 | AI 工作流
- **三種模式**：串聯型 / 並聯型 / 循環型
- **切入原則**：重複性 + 資訊處理 = AI 最佳節點
- **工具組合**：AI 對話層 + 自動化層 + 資料層 + 輸出層

### 第 4 堂 | 職場實戰
- **五大場景**：溝通 / 分析 / 創作 / 學習 / 決策
- **快速勝利**：把最耗時的重複工作先 AI 化
- **心態調整**：AI 優先思考，再評估是否人工

### 第 5 堂 | AI 工具製作
- **最快方法**：System Prompt 設計專屬 AI 助理
- **中階方法**：Make.com 串接自動化工作流
- **商業思路**：時間節省量 × 時薪 = 工具價值

### 第 6 堂 | 資料整理
- **四步驟**：Capture → Process → Organize → Review
- **核心 Prompt**：「摘要為：核心觀點、可行動洞察、相關問題」
- **首選工具**：Notion（靈活）/ Obsidian（深度）

### 第 7 堂 | 第二大腦
- **PARA**：Projects / Areas / Resources / Archives
- **AI 升級**：讓 AI 幫你分類、連結、提問
- **活化技巧**：每週回顧，讓 AI 找出知識關聯

### 第 8 堂 | 整合實戰
- **系統架構**：核心 AI → 自動化引擎 → 資料輸出
- **設計原則**：先手動驗證，再自動化
- **30 天計畫**：Prompt 庫 → 自動化 → 知識庫 → 評估優化

## Prompt 模板精選

```
【通用寫作】
你是一位 [領域] 專家。請幫我寫 [格式]，
目標讀者是 [對象]，重點包含 [要素]，語氣 [風格]，約 [字數]。

【資料分析】
以下是 [資料來源]，請分析：
① 3 個關鍵趨勢  ② 2 個異常點  ③ 1 個行動建議

【知識摘要】
請將以下內容整理成知識卡片：
標題 | 核心論點（3點）| 可行動洞察（2點）| 延伸問題（1個）

【決策分析】
我在考慮 [A] vs [B]，條件是 [限制]。
請分析優缺點，給出推薦，並說明最大風險。

【週回顧】
這週筆記：[清單]。請找出：共同主題、知識關聯、下週探索方向。
```
