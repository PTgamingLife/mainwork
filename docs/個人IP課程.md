## 第一段|開場 Hook:這個時代,沒有個人 IP 的人會怎樣?

> **講法**:一句話開場,不要自我介紹(這是我上課第一條規矩,也是我拍片第一條規矩——開頭自介 = 直接被滑掉)。

### Hook 講稿(0–60 秒抓住全場)

> 「我先問你一個問題。
> 十年前,你想加薪,要去求主管、求老闆、求那一間公司給你機會。
> 現在——**你可以直接讓一萬個陌生人替你決定你值多少錢。**
>
> 差別在哪?差別在你有沒有『個人 IP』。」

### 為什麼「沒有 IP」在這個時代等於被隱形

丟給學員三個扎心的現實(邊講邊點名他們身上正在發生的事):

1. **你的專業,別人看不到,就等於不存在。**
   同樣會做一件事的人太多了。市場不是付錢給「會做的人」,是付錢給「被看見、被信任的那個人」。沒有 IP,你永遠在跟全世界比價格;有 IP,別人是指名找你。

2. **演算法已經取代了履歷。**
   以前你的價值寫在一張 A4 履歷上,由一個 HR 決定。現在你的價值寫在你的內容裡,由演算法幫你分發給對的人。**不經營內容,就是把這個免費的分發引擎讓給別人用。**

3. **AI 讓「會做事」變便宜,「被信任的人」變更貴。**
   這是我這個帳號一直在講的:AI 出來以後,單純「會操作工具」的人一抓一大把,價值往下掉。真正漲價的,是**那個讓大家願意聽他說『這個工具怎麼用』的人**。技能會被 AI 稀釋,信任不會。IP 就是在囤積信任。

### 把痛講到底(留一個缺口,不要把話講死)

> 「所以最慘的不是你沒紅。
> 最慘的是——**你其實很有料,但這個世界不知道你存在,於是你只能拿跟『沒料的人』一樣的薪水。**
>
> 而打造 IP 這件事,門檻從來沒有像今天這麼低過。我一個英文普普、不會寫扣、要顧小孩、常常躲在廁所錄影片的爸爸都能做到——接下來這堂課,我把整套方法拆給你。」

> 🎯 **講師心法**:這一段的目的不是嚇人,是讓學員產生「我不能再拖了」的急迫感。痛點一定要配一個「但有解」的承諾(這是我踩過的坑:只丟痛點、不給出口,觀眾會直接關掉)。

## 第二段|IP 為什麼「一定」要有商業價值?

> **一句話定調**:「沒有商業價值的 IP,不是資產,是興趣。興趣會消耗你,資產會回饋你。」

### 先打破一個幻覺:粉絲數 ≠ 錢

我用自己的真實數據講給學員聽,不藏:

- 我看過**別人**一支「Claude 使用段位」的影片,26.4K 讚、3K 留言。
- 我也看過一則號召大家秀作品的貼文,衝到 **20.3 萬瀏覽**。
- 但我自己帳號很多貼文,是**個位數瀏覽**——不是內容爛,是帳號還沒過「冷啟動閘門」(這個第四段會拆)。

**重點是**:流量會騙人,讚數會騙人。**能不能換成錢,才是 IP 的體檢報告。**
一個有 5000 鐵粉、每個都願意買你東西的帳號,價值遠勝一個 10 萬泛粉、沒人掏錢的帳號。

### 為什麼「一定要有商業價值」——三個理由

1. **商業價值是 IP 的過濾器,逼你講對的話給對的人。**
   當你問自己「這篇內容能不能帶來會付錢給我的人」,你就不會再追那種「會爆但帶錯人」的題目。
   👉 我的血淚教訓:育兒補助、勞權、詐騙查詢這種題目,在台灣共鳴超強、很容易爆——但爆了帶來的是「找補助、找法條的人」,**不是想學 AI 的人**。這種粉絲越多,演算法越搞不清楚我是誰,反而害我。**會爆但帶錯受眾的題目,是傷害,不是紅利。**

2. **商業價值讓你撐得久。**
   純靠愛發電的帳號,九成會在「沒回饋」的疲乏裡默默停更。有現金流的 IP,才有理由每天在場、持續產出。

3. **商業價值決定你 IP 的『天花板』。**
   沒有變現路徑的 IP,天花板就是「多一點虛榮」;有變現路徑的 IP,天花板是「多一份、甚至取代本業的收入」。

### 商業價值不等於「馬上開課賣東西」

跟學員講清楚,商業價值有很多層,由淺到深:

| 層次 | 形式 | 什麼時候做 |
|------|------|-----------|
| 信任資產 | 有人因為你的內容記住你、信任你 | 一開始就在累積 |
| 導流價值 | 你能把流量導去某個地方(自己的作品、別人的活動) | 有一點聲量就能做 |
| 服務/接案 | 有人私訊「你能不能幫我做…」 | 展示過實作能力後 |
| 產品/課程 | 把你的方法打包成可複製的東西 | 方法被驗證、有人要之後 |

> 🎯 **講師心法**:我自己就是反「一紅就急著賣課割韭菜」的。**先給價值、把信任囤厚,商業價值是水到渠成的副產品,不是一開始就伸手要錢。** 這也是我帳號的正義敘事:官方免費就能學的東西,我不會包裝成幾千塊賣你。

## 第三段|如何打造個人 IP:一套可複製的定位公式

> **開場**:「打造 IP 不是『找到你有多特別』,是『找到一個你能持續講、市場又要聽的交集』。」

### 3-1 定位公式:IP = 領域 × 人設 × 受眾的痛

用我自己帳號當活教材,直接拆給學員看:

- **領域(你長期只講這件事)**:AI 工具實戰 / AI 變現。
  👉 為什麼要窄?因為**演算法是靠興趣推薦找受眾的**。你主題集中在 1–2 個領域,演算法才認得出「你是誰」、才找得到對的人。什麼都講 = 演算法搞不清楚你,誰都不推。

- **人設(你是誰、為什麼是你在講)**:工程師奶爸、英文普普、不會 code 也能手搓出東西、常常在廁所錄影片。
  👉 注意:**人設是『風味』,不是『主題』。** 「爸爸/不會 code/英文普普」是讓人記住我、對我有共鳴的調味料;但我的主題永遠是 AI,不是育兒。這條分不清,帳號就會走鐘。

- **受眾的痛(你替誰解決什麼)**:想用 AI 提升自己、兼職變現的上班族/自學者;怕被課程割韭菜;不會寫程式但想手搓出能用的東西。

**三個交集鎖住 = 你的 IP 定位。**
讓學員當場填一句話:「我是那個幫【誰】,用【領域】,解決【什麼痛】的人。」

### 3-2 內容支柱:一個帳號固定 3–4 種內容輪流發

不要每篇都在想「今天發什麼」,先定好支柱,輪流上:

| 支柱 | 目的 | 我的例子 |
|------|------|---------|
| 痛點型 | 引起共鳴、被轉發 | 「你用了 AI 卻還是天天加班,問題在這」 |
| 教學/清單型 | 被收藏、建立專業 | 「哪個情境該用哪個 AI:工具懶人包」 |
| 觀點型 | 表態、拉留言 | 「做 AI 變現,根本不用會寫程式」 |
| 展示型 | 證明你真的會 | 「我用 AI 手搓了一個小工具,過程給你看」 |

> 🎯 **講師心法**:內容要「輪替」,不要連發同一型。而且**每一篇都要能用一句話回答:『觀眾看完拿到什麼?』** 我踩過最痛的坑就是發「日記型心得文」——我多興奮、我怎麼用,結果 24 讚 0 留言。因為觀眾看完拿不到東西。心得類素材一律翻成「你可以怎麼做」的觀眾視角。

### 3-3 打造 IP 的三個階段(給學員一個時間感)

1. **養帳號期(冷啟動)**:主題集中、每天在場互動、靠留言別人的熱門貼文累積初始受眾。**這時候別期待單篇爆,先讓演算法認得你的 niche。**
2. **累積信任期**:穩定產出有用內容,讓人「因為有用而追蹤」,不是「因為你的故事而追蹤」。備 1–2 個能秀的小作品(我都用 Claude Code 手搓)。
3. **變現期**:有人開始私訊你、指名你,這時候商業價值自然浮現。

> 🎯 **反雞湯提醒**:別在 0 粉的時候幻想單篇破圈。我用真實數據驗證過——**同一套內容公式,頭部帳號拿 3.4 萬瀏覽,我 0 粉帳號只拿 4 瀏覽。差別在帳號,不在文案。** 打地基的階段騙不了人。

## 第四段|IG 如何創造流量:底層邏輯與平台規則

> **開場**:「先講一句最反直覺的話:**IG 的目標不是讓你紅,是讓用戶一直滑下去。你要做的,是變成那個讓人停不下來的內容。** 搞懂它想要什麼,它就幫你分發。」

### 4-1 底層邏輯:2026 演算法真正在乎什麼

**權重排序(這是重點中的重點,寫黑板):**

> **完播率 / 觀看時間 ≈ 35% > 收藏 25% > DM 轉發 20% > 讚 只剩 5%**

翻成人話跟學員講:

- **你的影片目標不是「被讚」,是「被看完、被存起來、被傳給某個人」。**
- **讚已經沒什麼用了。** 一支 100 個讚、沒人看完的影片,輸給一支 20 個讚、但大家都看完還轉發的影片。

**五條你必須刻進骨子裡的機制:**

1. **第 3 秒是生死線。** 大多數人滑過 3 秒還在看 → 演算法判定「這鉤子有效」,開始往外推。**所以開頭 3 秒定生死。**
2. **重播 = 超高價值訊號。** 同一個人看兩次,權重極高。短影片(7–15 秒)配「結尾接回開頭」的循環設計,最容易騙到重播。
3. **DM 轉發權重是讚的 3–5 倍。** 「看到想傳給某個人」的內容天生贏。所以要設計「傳給你那個○○的朋友」。
4. **發布後 30–60 分鐘的互動速度**,決定演算法要不要把你推給非粉絲(進 Explore / Reels 分頁)。所以發完文要守留言區。
5. **興趣推薦取代粉絲導向。** 主題集中,演算法才找得到對的人。Reels 觸及大約是照片的 2.5 倍。

### 4-2 鉤子怎麼設計:三層同時進攻

跟學員強調——**很多人關聲音在看(85%)**,所以你的 hook 不能只靠「講」:

- **口白層**:第一句就是鉤子,而且含關鍵字(IG 站內搜尋 + Google 都吃)。
- **畫面層**:第一幀就要有動作 / 結果 / 強烈畫面。**先秀成果,再講過程。靜止開頭是死刑。**
- **字卡層**:3–7 個大字壓縮你的承諾,高對比、放安全區。

**幾個我驗證過好用的鉤子公式**(讓學員抄走):
- 段位/等級對照:「用 AI 的 6 個段位,看看你在哪一級」——觀眾為了定位自己會看完、想升級會收藏、想表態會留言,**一個框架同時吃完播、收藏、留言三個機制**(這招我看到別人跑出 26.4K 讚)。
- 負面警告:「不要再手動○○了,一天白白燒掉 2 小時」(痛點一定要配上「那怎麼辦」的承諾)。
- 反直覺斷言:「做 AI 變現,根本不用會寫程式」。
- 秘密內幕:「99% 的人都不知道這個 AI 功能」。

### 4-3 平台規則:這些事一做就被降權(直接列清單)

> 「賺流量之前,先別被扣分。以下每一條我都用血換過:」

- ❌ 開頭自我介紹、鋪陳背景(「大家好我是…」= 直接被滑)
- ❌ 帶浮水印的轉載影片(TikTok logo 直接降權)
- ❌ 畫質差、非 9:16 直式(必須 1080×1920)
- ❌ 「按讚訂閱分享」連環 CTA(一支影片只下一個 CTA)
- ❌ 硬塞 30 個 hashtag(1–3 個精準標籤就好)
- ❌ 音樂蓋過口白、沒上字幕(關聲音看不懂 = 滑走)
- ❌ 把答案在 hook 就講完(主體沒理由留住人)

### 4-4 我自己踩過的三個「真實流量教訓」(這段最有說服力)

用我帳號的後台數據講,學員會信:

1. **「6–7 秒魔咒」——最大的瓶頸是節奏,不是題材。**
   我兩支影片題材完全不同、略過率差很多,但平均觀看時間都卡在 6–7 秒。原因是**全程單鏡口播、零視覺變化**。
   👉 解法:**主體每 2–3 秒就要一個視覺變化**(跳剪 / zoom / 換角度 / 字卡進場 / B-roll)。靜止超過 4 秒 = 滑走高危區。

2. **「猛料一定要前置到 0–3 秒」。**
   我曾把最有價值的東西(免費資源)埋到影片第 23 秒 → 4 天 8 個讚,躺平。
   👉 解法:先找出素材裡「你最捨不得先講的那張牌」,直接放第一句。

3. **「留言關鍵字機制」——把留言率從 1–3% 拉到 35%。**
   資訊型內容大家願意收藏、但懶得留言。但只要給一個交換條件——「留言『○○』,我把完整教學私訊給你」——留言率可以衝到 35%(有帳號跑出 2.8K 讚 / 991 則留言)。
   👉 前提:(a) 你手上真的有東西可以給(模板/清單/資源);(b) 關鍵字要短、跟主題同字。

### 4-5 冷啟動閘門:為什麼新手「發了沒人看」是正常的

這段一定要講,不然學員會誤會自己失敗:

> **「演算法先看『這是誰發的』,再決定要給你多少初始流量。」**
> 個位數瀏覽 ≠ 你內容爛,是你帳號還沒過冷啟動閘門。這時候**改文案救不了**,要做的是:
> - 主題集中在 1–2 個領域(讓演算法認得你的 niche)
> - 每天在場,認真回別人的熱門貼文(回覆的價值 ≈ 自己發文)
> - 一週穩定發 3–5 支,勝過爆發連發再消失

**診斷公式(送給學員的體檢表):**
- 觀看低 + 互動也低 → 帳號冷啟動問題(養帳號,別急著改文案)
- 觀看高 + 互動低 → 題材對、包裝錯(修 hook、修節奏、加互動缺口)
- 觀看低 + 互動高 → 包裝對、選題或標籤錯(修選題、修標籤)

## 第五段|七天作業:接下來 7 天,打造你 IP 的第一塊地基

> **交代方式**:「聽完不做,這堂課等於沒上。這七天不是叫你爆紅,是叫你把地基打下去。每天 30–60 分鐘,做完你就有一個能開始跑的 IP 雛形。」

### Day 1|定位——把你是誰講成一句話
- 用第三段的公式,寫出:「我是那個幫【誰】,用【什麼領域】,解決【什麼痛】的人。」
- 選定你的**唯一領域**(1–2 個,不能更多)。
- 寫下你的**人設風味**(3 個關鍵字,例如我的:工程師 / 奶爸 / 英文普普)。
- ✅ 交付:一句定位 + 一個領域 + 三個人設關鍵字。

### Day 2|受眾與痛點——列出你要服務的人
- 寫出你的目標受眾「一天當中,跟你領域有關的 3 個具體煩惱」。
- 每個煩惱後面寫「他現在都怎麼(笨笨地)解決」。
- ✅ 交付:3 個真實痛點 + 現況,這就是你未來的選題礦。

### Day 3|內容支柱——排出你的 4 種內容
- 用第三段的表,為自己的帳號定 3–4 個內容支柱(痛點 / 教學 / 觀點 / 展示)。
- 每個支柱先想 2 個題目 → 湊出 8 個選題,放進你的「題庫」。
- ✅ 交付:一張內容支柱表 + 8 個備用選題。

### Day 4|帳號基建——把個人頁整理好
- 把 bio 寫到「陌生人 3 秒內就懂你是誰、追蹤你能得到什麼」。
- 頭像清楚、帳號名稱含關鍵字。
- 準備 1 個「能秀的小作品/成果」當作展示素材(不用大,能證明你會就好)。
- ✅ 交付:改好的個人頁截圖。

### Day 5|寫出第一支腳本——套三層 Hook
- 從 Day 3 題庫挑 1 個題目,寫成一支 15–30 秒腳本。
- 必須包含:①口白層鉤子(第一句含關鍵字)②畫面層(第一幀秀成果)③字卡層(3–7 字)。
- 檢查:主體有沒有「每 2–3 秒一個視覺變化」?結尾有沒有「一個 CTA」?
- ✅ 交付:一份完整腳本(Hook / 主體 / 收尾三段)。

### Day 6|發佈 + 守留言區——實測冷啟動
- 把 Day 5 的腳本拍出來發佈(9:16、上字幕、1 個 CTA、1–3 個精準標籤)。
- **發佈後 60 分鐘守在留言區**,快速回覆衝互動速度。
- 同一天:去你領域裡一則熱門貼文的留言區,留一則高質量、有料的留言(借流量、被看見)。
- ✅ 交付:發佈連結 + 留言互動截圖。

### Day 7|複盤——用診斷公式讀數據,不看讚看對的指標
- 打開後台,記下:觀看數、平均觀看時間、略過率、收藏、分享。
- 用第四段的**診斷公式**判斷:你現在的問題是「帳號冷啟動」還是「內容包裝」?
- 寫下:下一支要改的**一件事**(只改一件,別全改)。
- ✅ 交付:一張數據複盤表 + 下一步的一個改進點。
